# iList前端
## 技术栈
[react](https://react.dev/)

JavaScript
## 项目介绍
一款支持云存储的文件列表程序
## 基本功能
- [x] 上传文件或文件夹到云端
- [x] 从云端下载文件至本地
- [x] 拥有文件夹系统 
## 本地部署与运行
1. 克隆到本地
```
git clone https://github.com/BGboy-bit/ilist-forend.git
cd ilist-forend
```
2. 安装依赖
```
npm i  
```
3. 运行
```
npm start
```
4. 构建
```
npm run build
```
